# Reports — Executive Assistant System

**Module:** Executive Assistant System  
**Last updated:** 2026-04-22

---

## Purpose

This directory stores all output files generated by Executive Assistant System operations. All files here are DRAFT status unless explicitly marked otherwise.

---

## File Naming Convention

```
[YYYY-MM-DD]-[report-type]-[scope]-[version].md

Examples:
  2026-04-22-outlook-audit-business-v1.md
  2026-04-22-finance-snapshot.md
  2026-04-22-contact-candidates.csv
  2026-04-22-pilot-briefing.md
```

---

## Report Types

| Type | Source prompt / process | Output format |
|------|------------------------|--------------|
| `outlook-audit` | `READ_ONLY_OUTLOOK_AUDIT_PROMPT.md` | Markdown |
| `finance-snapshot` | Finance module extraction | Markdown |
| `contact-candidates` | Contact extraction | CSV + Markdown summary |
| `pilot-briefing` | Briefing Expansion Plan | Markdown |
| `system-verification` | `FINAL_SYSTEM_COMPLETION_PROMPT.md` | Markdown |

---

## Status Labels

Every report file must include one of these status labels in its header:

| Label | Meaning |
|-------|---------|
| `DRAFT` | Generated — not yet reviewed |
| `REVIEWED` | Operator has read and noted observations |
| `APPROVED` | Operator has confirmed output is accurate |
| `ARCHIVED` | No longer current — retained for reference |

---

## Retention Policy

- DRAFT reports: retained for 30 days unless promoted to APPROVED
- APPROVED reports: retained until superseded by a newer version
- Reports must not be deleted without operator decision — mark as ARCHIVED first
